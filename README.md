This project is written in Go and consists of four tasks.
Each task is implemented in a separate package.
The project demonstrates the use of structs, interfaces, maps, slices, and methods.
The main file imports all packages and runs their logic.


---

## Task 1: Hotel Room Reservation System
**Package:** `Assignment1/Hotel`

- Manages hotel room bookings
- Uses `Room` and `Hotel` structs
- Stores rooms in a map
- Supports adding rooms, check-in, check-out, and listing vacant rooms

---

## Task 2: Employee & Interfaces
**Package:** `Assignment1/Employee`

- Implements an `Employee` interface
- Supports FullTime, PartTime, Contractor, and Intern employees
- Each employee type has its own salary and bonus calculation logic
- Employees are stored in a slice and processed using a loop

---

## Task 3: Gym Membership Management System
**Package:** `Assignment1/Gym`

- Implements a `Member` interface
- Supports BasicMember and PremiumMember
- Uses a map to store gym members
- Allows adding and listing members

---

## Task 4: Wallet Transaction Simulation
**Package:** `Assignment1/Wallet`

- Simulates a digital wallet
- Supports adding and spending money
- Stores transaction history in a slice
- Displays current balance
